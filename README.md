# git-interface

git多仓库管理

## 使用说明

* 入参数量小于2时，读取同目录下的配置文件
* 入参数量大于等于2时：
    * 第一个入参：仓库配置文件路径
    * 第二个入参：分组配置文件路径

## 功能

* 批量 fetch
* 批量 rebase
* 分组功能
